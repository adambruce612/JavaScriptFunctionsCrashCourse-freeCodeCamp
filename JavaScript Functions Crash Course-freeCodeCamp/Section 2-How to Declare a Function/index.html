How to Declare a Function

Functions do not always have to have names.

An example function: function printMe() { console.log('printing...') }

After declaring a function you can call it with the following: printMe(); - this will execute the code that was written in the function

Functions can be set up to take parameters, ex: function printThis(param) {console.log(param);}.

The value that is passed into the function is called an argument.

You can create a function as an expression. Example: const printMe = function() { console.log('print'); }
